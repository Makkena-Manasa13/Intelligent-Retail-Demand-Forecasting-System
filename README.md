# Intelligent-Retail-Demand-Forecasting-System
AI Retail Analytics Dashboard is a full-scale intelligent retail management system designed to analyze product sales, monitor inventory, and predict future demand using machine learning-driven analytics.

Features
Modern dark-themed enterprise dashboard
Product-wise analytics with detailed click view
Inventory monitoring and stock alerts
Sales trend visualization
Forecasting dashboard for future demand
Product image integration
Category-based filtering and search
AI-powered recommendations for stock planning
Low stock and out-of-stock alerts
Revenue and category performance insights
AI Capabilities

The system uses predictive analytics to:

forecast product demand
detect high-selling products
identify stock shortages
generate smart restock recommendations

pip install -r requirements.txt
python run.py

Frontend

cd frontend
npm install
npm start
